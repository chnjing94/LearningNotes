## 线程模型

**Redis 真的只有单线程吗？**

不是，Redis 单线程是指它对网络 IO 和数据读写的操作采用了一个线程；但 Redis 的其他功能，比如持久化、异步删除、集群数据同步等，其实是由额外的线程执行的。

**为什么单线程的 Redis 能那么快？**

1. 读写操作都在内存中
2. 高性能网络I/O模型
3. 避免了多线程并发控制带来的开销
